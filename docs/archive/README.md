# Архив документации

Эта папка содержит архивную документацию проекта:

- **CLEANUP_SUMMARY.md** - отчет об очистке проекта
- **VIDEO_ANALYSIS_REPORT.md** - анализ активности движений и тестирование на шуме
- **FINAL_RESULTS.md** - финальные результаты улучшенной модели
- **NORMALIZATION_COMPARISON.md** - сравнение методов нормализации
- **FIX_SUMMARY.md** - сводка исправлений
- **MODEL_ANALYSIS.md** - анализ архитектур моделей
- **DIAGNOSTIC_ANSWERS.md** - ответы на диагностические вопросы
- **TESTING_GUIDE.md** - руководство по тестированию
- **training_summary.md** - сводка обучения
- **TRAINING_README.md** - документация по обучению

Эти файлы сохранены для справки и истории разработки проекта.
